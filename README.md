Featured Project: Loan Portfolio & NPE Analysis
Objective

Analyze a loan portfolio to assess performance, identify non-performing exposure (NPE), and uncover risk patterns across customer segments.

Key Questions Answered
What is the trend of non-performing exposure over time?
Which customer segments (e.g., age groups) carry higher risk?
How does portfolio quality evolve across reporting periods?
Methodology
Data cleaning and validation
Feature engineering (e.g., age groups, delinquency flags)
Aggregation and metric computation (NPE rate, balances)
Time-series and segment-level analysis
Visualization for stakeholder interpretation
Sample Metrics
NPE Rate (≥ 30 days past due)
Total Exposure vs Non-Performing Exposure
Segment-level risk distribution
Tools & Technologies
Python (Pandas, NumPy)
Data Visualization (Matplotlib, Seaborn)
Jupyter Notebooks
SQL (where applicable)
